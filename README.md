# Discord Voice & Video Fix (RU/CIS)
Утилита для восстановления работоспособности голосовых каналов и обхода замедления. 
Работает на уровне драйвера, не требует настройки прокси.

## Инструкция:
1. Скачайте архив из раздела Releases.
2. Запустите DiscordFix_Setup.exe от имени администратора.
3. Нажмите "Apply Patches".
